Overview

Purpose

The purpose of my website is to bring traditional Brazilian dishes. I won't show the recipe, I only will display the dishes. Much more like a galery of brazilian dishes.

Audience

The audience I want to reach is diverse. I aim to connect with Brazilians themselves as well as people who have had some form of contact with Brazil, for example: visited Brazil and liked the food, married someone from Brazil, served a mission in Brazil, etc.

Dynamic elements

Mobile Menu Toggle: JavaScript will be used to create an interactive mobile menu. When users click the mobile menu button (#mobile_btn), JavaScript will toggle the visibility of the mobile menu (#mobile_menu) by adding or removing the active class. This will allow the menu to slide in and out of view. Additionally, the icon inside the button will change to indicate whether the menu is open or closed. Dynamic Header Shadow: JavaScript will add a dynamic shadow to the header as users scroll down the page. When the scroll position is greater than the height of the header, a subtle box-shadow will be added to the header to give a sense of depth and separation from the rest of the content. This visual cue will enhance the user experience by making the navigation bar stand out. Active Section Highlighting: As users scroll through the page, JavaScript will highlight the corresponding navigation item (.nav-item) based on the section currently in view. This will help users keep track of their location on the page. The active section will be determined by comparing the scroll position to the positions of the sections, and the active class will be applied to the appropriate navigation item. ScrollReveal Animations: JavaScript will use the ScrollReveal library to add engaging animations to various elements on the page. As users scroll, elements such as the call-to-action section (#cta), dish elements (.dish), testimonial chef section (#testimonial_chef), and feedback elements (.feedback) will animate into view. These animations will originate from different directions and have varying durations, creating a visually appealing and dynamic experience.
